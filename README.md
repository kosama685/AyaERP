## ⚡ Features

### 📊 Live Dashboard (Streamlit)
- Corporate dark theme with glass-morphism UI
- Real-time KPI metrics across all clients
- Client portfolio with regional flags
- Quick action buttons for daily automation

### 🤖 AI Router — Token Optimizer
- Routes tasks to the most cost-effective AI model
- **ChatGPT Pro** → Short-form content, auto-replies
- **Claude Pro Max** → Long-form strategy, lead scoring
- **Grok** → Real-time trends, competitor news
- **Gemini Pro** → Images, video, SEO/AEO/GEO
- **~$10/month** total AI cost for 4 clients

### 🎯 Lead Detection Engine
- Auto-scores messages/DMs for sales intent (0-100)
- 🚨 HIGH (60+) → Immediate WhatsApp/Email alert
- ⚡ MEDIUM (30-59) → Auto-reply + CRM
- 💬 LOW (<30) → Auto-reply only
- Supports **Arabic** 🇸🇦 and **Urdu** 🇵🇰 keywords

### 🎪 Exhibition Database
- 25+ exhibitions across 5 regions
- Filterable by niche (tech, hospitality, marketing, ecommerce)
- Upcoming event tracking with reminders

### 📅 Best Posting Times
- Region-specific schedules (SA, UK, EU, US, PK)
- Platform-specific best times (IG, LinkedIn, FB, X, TikTok)
- Monthly content themes (Ramadan, LEAP, Hajj, Black Friday)

### 📋 Strategy Generator
- 12-month marketing plans per client
- Platform scoring by region + industry
- Budget allocation with progress visualization
- SEO/AEO/GEO keyword strategy
- KPI targets and measurement framework

### 🔄 6-Phase ERP Automation
1. **Data Input** → Client onboarding, market research
2. **AI Strategy** → Plans, platform scoring, exhibitions
3. **Content Factory** → AI generation, multi-language
4. **Distribution** → Smart scheduling, auto-posting
5. **Engagement** → Auto-reply, CRM, lead detection
6. **Reporting** → Selenium scrape, dashboards, email

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/kosama685/AyaERP.git
cd AyaERP

# Install dependencies
pip install -r requirements.txt

# For Selenium features (optional)
pip install selenium webdriver-manager

# Run the Streamlit dashboard
streamlit run app.py

# Or run the automation engine
python3 scripts/erp_engine_v2.py run
```

## 🌐 Deploy on Streamlit Cloud

1. Go to [streamlit.io/cloud](https://streamlit.io/cloud)
2. New App → Select `kosama685/AyaERP`
3. Branch: `main` · Main file: `app.py`
4. Deploy! Live in ~2 minutes
